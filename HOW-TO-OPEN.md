## 📝 Step-by-Step Guide

### Step 1: Start Backend
1. Open PowerShell
2. Navigate to backend folder:
```bash
cd C:\Users\vasum\Desktop\Hackathan12\backend
```
3. Run the setup script:
```bash
.\setup-and-run.ps1
```
4. Wait for: `Started GamifiedLearningApplication`

### Step 2: Open Website
**Method 1 - File Explorer (Easiest)**:
1. Open File Explorer
2. Go to: `C:\Users\vasum\Desktop\Hackathan12\frontend`
3. Right-click on `index.html`
4. Select "Open with" → "Google Chrome"

**Method 2 - Chrome Directly**:
1. Open Google Chrome
2. Press `Ctrl + O`
3. Navigate to: `C:\Users\vasum\Desktop\Hackathan12\frontend\index.html`
4. Click "Open"

**Method 3 - Drag and Drop**:
1. Open Google Chrome
2. Open File Explorer to: `C:\Users\vasum\Desktop\Hackathan12\frontend`
3. Drag `index.html` into Chrome window

## 🐛 Troubleshooting

**Problem**: Whitelabel Error
**Solution**: Don't use `http://localhost:8081/` - Open the HTML file directly!

**Problem**: Backend not starting
**Solution**: Check Java 17 is installed and MySQL is running

**Problem**: Features not showing
**Solution**: Make sure you're opening `index.html` (not index-v2.html)
