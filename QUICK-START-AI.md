# 🚀 Quick Start - AI Generator & Real-Time Feedback

## ⚡ 5-Minute Setup

### Step 1: Start Backend (30 seconds)
```bash
cd backend
.\setup-and-run.ps1
```

Wait for: `Started GamifiedLearningApplication`

### Step 2: Open Application (10 seconds)
Open browser and go to:
```
http://localhost:8081/
```

Or open: `frontend/index-v2.html`

### Step 3: Login (20 seconds)
- Click "Register" tab
- Username: `yourname`
- Email: `your@email.com`
- Password: `password123`
- Click "Register"

### Step 4: Try AI Generator (2 minutes)
1. Click **"AI Generator"** button in navigation
2. Select:
   - Category: **Mathematics**
   - Difficulty: **Easy**
   - Topic: Leave blank or type "Addition"
3. Click **"✨ Generate Question"**
4. Review the question
5. Click **"💾 Save to Question Bank"**
6. Success! ✅

### Step 5: Test Real-Time Feedback (2 minutes)
1. Click **"Dashboard"** button
2. Click on **"Mathematics"** category card
3. Answer a question (any option)
4. Watch the magic:
   - ✅ Green animation if correct
   - ❌ Red shake if incorrect
   - 🎊 Confetti effect
   - 🔊 Sound effect
   - 💬 Encouragement message
   - 📊 Points update

## 🐛 Quick Fixes

### Backend won't start?
```bash
# Check Java
java -version

# Should show: java version "17.0.12"
```

### Can't connect to database?
```bash
# Check MySQL
mysql -u root -p
# Password: Laji@05#02
```

### Questions not appearing?
```
1. Refresh page (F5)
2. Clear cache (Ctrl+Shift+Delete)
3. Check backend logs
```
