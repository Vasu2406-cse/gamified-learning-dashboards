## 🚀 How to Use

### Start Backend:
```bash
cd backend
.\setup-and-run.ps1
```

### Open Application:
```
http://localhost:8081/
```
Or open: `frontend/index-v2.html`

### Generate Questions:
1. Login/Register
2. Click "AI Generator"
3. Select category, difficulty, topic
4. Click "Generate Question"
5. Click "Save to Question Bank"

### See Feedback:
1. Click "Dashboard"
2. Start any quiz
3. Answer questions
4. Watch confetti, animations, sounds!
