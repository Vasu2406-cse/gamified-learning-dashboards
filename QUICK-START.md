## 🔧 Troubleshooting

### Website not loading?
- Make sure backend is running on port 8081
- Refresh the browser (Ctrl + F5)

### No questions showing?
- Check if you're logged in
- Try selecting a different category

### Backend not running?
- Check if port 8081 is free
- Restart the backend server
